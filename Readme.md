# Simple Calculator – Python

## 📌 Project Description

This project is a **simple calculator application built using Python**.
It allows users to perform basic arithmetic operations such as **addition, subtraction, multiplication, and division** by entering two numbers and selecting the desired operation.

This project was developed as part of an **Internship Task** to demonstrate basic programming concepts like:

* User input
* Conditional statements
* Arithmetic operations
* Error handling

---

## ⚙️ Features

* Accepts two numbers from the user
* Allows selection of arithmetic operation
* Supports the following operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Handles division by zero errors
* Displays the result clearly to the user

---

## 🛠️ Technologies Used

* **Python 3**
* Command Line Interface (CLI)

## ▶️ How to Run the Program

1. Install **Python 3** on your system.
2. Download or clone the repository.
3. Open the project folder in **VS Code or Terminal**.
4. Run the following command:

```
python Calculator.py
```

5. Enter the numbers and choose the operation when prompted.
