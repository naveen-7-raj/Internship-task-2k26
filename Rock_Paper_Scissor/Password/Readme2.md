# Password Generator – Python

## 📌 Project Description

This project is a **Password Generator application built using Python**.
The program generates a **strong and random password** based on the length specified by the user.

The goal of this project is to demonstrate basic Python programming concepts such as **user input, random generation, and string manipulation**.

This project was developed as part of an **Internship Programming Task**.

---

## 🔐 Features

* User can specify the **desired password length**
* Generates a **random and strong password**
* Uses a combination of:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* Displays the generated password instantly

---

## 🛠️ Technologies Used

* **Python 3**
* Built-in Python modules:

  * `random`
  * `string`

## ▶️ How to Run the Program

1. Install **Python 3** on your system.
2. Clone or download this repository.
3. Open the project folder in **VS Code or Terminal**.
4. Run the program using:

```
python password_generator.py
```

5. Enter the **password length** when prompted.
6. The program will generate and display a **random secure password**.
