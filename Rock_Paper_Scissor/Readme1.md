# Rock-Paper-Scissors Game – Python

## 📌 Project Description

This project is a **Rock-Paper-Scissors game implemented in Python**.
The user plays against the computer, which randomly selects **rock, paper, or scissors**. The program determines the winner based on the standard game rules.

This project was created as part of an **Internship Programming Task** to demonstrate basic Python programming skills.

---

## 🎮 Game Rules

* **Rock beats Scissors**
* **Scissors beats Paper**
* **Paper beats Rock**
* If both the user and the computer choose the same option, the result is a **tie**.

---

## ⚙️ Features

* User chooses **rock, paper, or scissors**
* Computer generates a **random choice**
* Displays both selections
* Determines and displays the **winner**
* **Score tracking** for user and computer
* Option to **play multiple rounds**

---

## 🛠️ Technologies Used

* **Python 3**
* `random` module for computer choice
* Command Line Interface (CLI)

## ▶️ How to Run the Program

1. Install **Python 3** on your system.
2. Download or clone this repository.
3. Open the project folder in **VS Code or Terminal**.
4. Run the program using:

```
python rock_paper_scissors.py
```

5. Enter your choice (**rock, paper, or scissors**) when prompted.
